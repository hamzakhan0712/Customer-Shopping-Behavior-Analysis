### Deliverables

1. **Data Preparation & Modeling (Python)**  
   Clean and transform the raw dataset to make it analysis-ready.

2. **Data Analysis (SQL)**  
   Structure the data, simulate business transactions, and run SQL queries to generate insights on customer segments, loyalty, and purchase drivers.

3. **Visualization & Insights (Power BI)**  
   Develop an interactive dashboard that highlights key trends and patterns to support data-driven decision-making.

4. **Report and Presentation**  
   Prepare a clear project report summarizing findings and business recommendations, and create a presentation that communicates insights and actionable strategies to stakeholders.

5. **GitHub Repository**  
   Maintain a well-structured repository containing all Python scripts, SQL queries, and dashboard files.
